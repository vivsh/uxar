Dev-only stylebook

One visual system before every UI surface.

This page validates palette, typography, components, and product layouts before they move into landing, mdBook, or console.

Rules

Bootstrap composes. Vyuh defines.

Allowed

Use Bootstrap utilities for page composition: containers, rows, columns, gaps, flex, display, and section spacing.

Forbidden

Do not depend on utility classes inside reusable Vyuh components. Component shape, spacing, border, shadow, and typography live in SCSS.

Promotion rule

If a utility combination repeats or becomes required for correctness, promote it into a named semantic Vyuh class.

Scales and presets

Small scales, named decisions.

Use these presets before adding a new local value. If a component needs a new rhythm, add it to the scale first.
